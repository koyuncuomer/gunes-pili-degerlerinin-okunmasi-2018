# Güneş Pili Değerlerinin Okunması
PIC 16F877A mikro denetleyicisi kullanılarak bir güneş pilinden gelen volt ve akım değerleri ölçülüp bu değerlerin LCD ekranda gösterilmesi amaçlanmıştır.

# Kullanılan malzemeler
- PIC 16F877A
- 6V Güneş Pili
- 16x2 LCD Ekran
- 4 Mhz Kristal Osilatör
- 2 adet 22pF Mercimek Kondansatör
- 3 adet 10K Direnç
- Erkek-erkek Jumper Kablo
- Breadboard
- Breadboard Güç Kartı
- Buton (eğer reset bağlantısı kurulacak ise kullanılacaktır, [mikro denetleyicinin reset bacağı ile direnç arasına yerleştirilecektir](https://www.mustafasolak.com/2016/10/29/gomulu-sistemler/13/pic16f877a-pinleri-ve-temel-baglantilar "detaylı bilgi için tıklayınız"))
- 10 kΩ potansiyometre ([LCD ekranın parlaklığını ayarlamak için kullanılacaktır](https://www.8051projects.net/t50417/pic-microcontroller-discussion/16x2-lcd-16f877a.htm "örnek bağlantı şeması için tıklayınız"))
